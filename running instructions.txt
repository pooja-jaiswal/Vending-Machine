Project executable(s) of the Vending Machine components 

1. Install Maven. Please follow below link to install Maven
	https://maven.apache.org/install.html
2. Open terminal/command prompt and go to the project folder (/vending-machine/code) where the pom.xml file is located.
3. Run below command:
	"mvn clean install"
4. A target folder will be created (in "/vending-machine/code" folder). Go into target folder. 
5. Inside the target folder, you will find "vending-machine.exe" and "vending-machine-1.0.jar" file.
6. Double click "vending-machine.exe" file to launch the application or run below command in terminal/command prompt to launch the application.
	"java -jar vending-machine-1.0.jar"
